# Assets Folder

This folder contains icon assets for the tab bar.

## Icon Files

All icons are 50x50 PNG files downloaded from Icons8:

- `home.png` - Home icon for Basic tab
- `palette.png` - Paint palette icon for Customized tab
- `bolt.png` - Lightning bolt icon for Auto-Fill tab
- `ruler.png` - Ruler icon for Sizes tab
- `warning.png` - Error/warning icon for Error tab

All icons use the app's primary color (#4A90E2) for consistency with the UI theme.

## Customization

To replace icons:
1. Use PNG format (50x50 or similar size)
2. Consider creating @2x and @3x versions for different screen densities
3. Keep icons simple and recognizable at small sizes
4. Maintain consistent style across all icons
